拼音快速練習

ㄅ

(不需輸入聲調)

快速對照表

學習重點

  1. 容易受英文發音干擾的字母
    • q 發 ㄑ 的音（不是 K 或 Kw）。
    • x 發 ㄒ 的音（不是 Ks）。
    • c 發 ㄘ 的音（不是 K 或 S）。
    • z 發 ㄗ 的音。
    • zh 發 ㄓ 的音。
  2. 隱藏的母音（縮寫規則）
    • ㄧㄡ (i+ou)： 寫成 iu（藏了 o）。例：九 = jiu。
    • ㄨㄟ (u+ei)： 寫成 ui（藏了 e）。例：對 = dui。
    • ㄨㄣ (u+en)： 寫成 un（藏了 e）。例：吞 = tun。
  3. 【關鍵陷阱】「ㄨㄣ」跟「ㄩㄣ」長得一模一樣 (un)
    • 唸 ㄩㄣ (ün)： 只要前面是 j (ㄐ)、q (ㄑ)、x (ㄒ)。
      (例：軍 = jun、群 = qun、尋 = xun)
    • 唸 ㄨㄣ (uen)： 只要前面是其他所有子音（如 g, k, h, zh, z, s 等）。
      (例：昆 = kun、準 = zhun、孫 = sun)
    • 自己獨立發音時： 如果前面沒有子音，它們就會變身來區別彼此：溫 = wen，暈 = yun。
